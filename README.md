# 📊 Stock Analysis

An exploratory data analysis of stock market data using Python.

## Overview
This project explores historical stock market data to analyze price trends, volatility, and performance metrics.

## Contents
- `Stock_A.ipynb` — Jupyter notebook with data exploration and analysis

## Tools Used
- Python
- Pandas / NumPy
- Matplotlib / Seaborn

## How to View
Open `Stock_A.ipynb` in Jupyter Notebook, JupyterLab, or Google Colab to run the analysis and view the results.
